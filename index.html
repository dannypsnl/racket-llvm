Racket LLVM

A racket LLVM C-API bindings, document will show you how to use this library as simple as possible.

 (require racket-llvm) package: racket-llvm

syntax

(define-llvm id function-type #:c-id c-function-id)

define-llvm takes an id as its produces function name, then is a C _fun defined types for C function. Finally, takes the link name of C function. Using define-llvm to extend any functions missing in this library will be important.
